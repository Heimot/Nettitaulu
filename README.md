# What is this application for?

I made this application to help with keeping track of orders while gathering them. 

# Installation

Use NPM package manager to install.

```bash
npm install
```

To update Node_modules

```bash
npm update
```

To run the application in your localhost

```bash
npm start
```

To build a production version of the application

```bash
npm install
```
